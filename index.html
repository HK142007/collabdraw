Collabdraw

HTML5 collaborative whiteboard for tablets

Download this project as a .zip file Download this project as a tar.gz file

SERVER REQUIREMENTS:

  1. Python 3.2+
  2. All python packages specified in requirements.txt (which might involve installing other non-python dependencies like cairo, redis)
  3. Redis server
  4. libpoppler, imagemagick for upload functionality
  5. ffmpeg to enable video functionality

INSTALLATION:

  1. Clone the git repository git clone git://github.com/anandtrex/collabdraw.git
  2. Initialize submodules git submodule init && git submodule update
  3. Set the hostnames, ports and other options in config.py

RUNNING:

  1. Run the redis server
  2. Run "python main.py"

KNOWN-ISSUES:

  1. Doesn't work very well on android gingerbread browsers due to limited html5 support
